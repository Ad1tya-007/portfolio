# macOS Portfolio

A beautiful portfolio website styled like macOS Monterey/Ventura.

## Features

- Welcome screen with start button
- Loading screen with Apple logo animation
- macOS-style desktop with menu bar
- Interactive folders for portfolio sections
- Glassmorphism dock with app icons
- Folder windows with authentic macOS styling

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Folders

- **About Me** - Personal information
- **Projects** - Portfolio projects
- **Skills** - Technical skills
- **Experience** - Work experience
- **Contact** - Contact information

## Tech Stack

- React 18
- Vite
- CSS3 (with glassmorphism effects)
